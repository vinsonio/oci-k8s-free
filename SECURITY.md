## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability in this project, please let us know right away.

We strongly encourage you to report security vulnerabilities privately so that we have time to fix them before they are known to the public. 

Please report security issues by using GitHub's **Private Vulnerability Reporting** feature. To do this:
1. Go to the **Security** tab of the repository.
2. Click on **Report a vulnerability**.
3. Provide a clear and detailed description of the vulnerability, including:
   - The type of vulnerability.
   - The location within the codebase.
   - Steps to reproduce the issue.
   - Any potential impact.

Upon receiving your report, maintainers will review the issue and work with you to understand and resolve it. We will try to confirm the vulnerability and provide a timeline for a fix as soon as possible.
